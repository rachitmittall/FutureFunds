# FutureFunds 💰

> Learn to invest before you earn.

## About
FutureFunds is a financial simulator for Indian 
college students. Simulate real-world investing 
with live market data before risking real money.

## Features
- 📊 Real-time prices (Stocks, Crypto, Gold, Silver, MF)
- 💸 City-based expense simulator
- 📈 5-year portfolio projection
- 🕰️ Time Machine — see what past investing looks like
- ⚡ Fast Forward — project wealth decades ahead
- 🤖 AI-powered portfolio insights
- 💾 Auto-saves your portfolio locally

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- Recharts
- Vercel Serverless Functions

## Getting Started
npm install
npm run dev
